# Olist Revenue Leakage Analysis

## Business Problem
Identifying product categories driving cancellations 
and quantifying revenue leakage.

## Key Findings
- PC Gamer: 11% cancellation rate — 17x platform average
- Total revenue lost: ₹95,235 across 2016–2018
- July 2018 spike: highest single-month loss at ~₹18K

## Tools Used
PostgreSQL | Python (Pandas) | PowerBI
